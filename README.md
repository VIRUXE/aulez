# AULEZ - Avoid Ultra Low Emissions Zone

Based on "Juliesmaps.co.uk"

## Legend

- **Red**: Live
- **Black**: Dead
- **Red "-"**: Live
- **Red "A"**: An ANPR camera and a ULEZ camera
- **Black "-"**: Dead
- **Other Black "X"**: Pole only, no camera yet
- **Black "C"**: Cable cut
- **Black "P"**: Painted / stickered
- **Black "M"**: Missing (stolen, etc.)
